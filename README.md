## 3-Month Roadmap

<details>
<summary>Month 01 — Foundations</summary>

**Focus:** Basics, syntax, understanding Go

**Topics:**

- Install Go, `go run`, `go build`, modules
- Variables, functions, loops, slices, maps
- Structs & methods, error handling

**Mini Projects:**

- Calculator CLI  
- To-do list CLI  
- File reader/writer  

</details>

<details>
<summary>Month 02 — Deep Dive + CLI</summary>

**Focus:** Real programs and DevOps-style tools

**Topics:**

- Interfaces & packages  
- JSON & YAML parsing  
- HTTP API requests  
- Environment variables  
- CLI apps with `cobra`  
- Logging & error handling  

**Mini Projects:**

- API CLI (e.g., weather app)  
- Config validator  
- Log parser  

</details>

<details>
<summary>Month 03 — Concurrency + DevOps</summary>

**Focus:** High-performance Go for DevOps

**Topics:**

- Goroutines & channels  
- Worker pools & mutex  
- Docker API  
- Kubernetes API  

**Final Projects:**

- Server monitoring tool  
- Kubernetes helper CLI  
- Docker container manager  
- Cloud automation script (AWS/GCP)  

</details>
